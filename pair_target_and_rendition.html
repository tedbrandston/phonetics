Input

A .csv file with two columns: target and rendition. You can export this from a google doc via File->Download as->Comma-Separated Values

The number of words (space-separated runs of letters) must match between the target and the rendition. If any mismatches are found they will be printed instead of the output.

Output

If errors are encountered processing the input, those will be printed. Otherwise an alphabetized list of <target word>, <source line number>, <rendition> for each word in the input targets.
